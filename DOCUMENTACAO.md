# Documentação Técnica

## Visão Geral
Este projeto é uma página simples em HTML que exibe um título, um texto e um botão.  
Ao clicar no botão, uma mensagem é mostrada na tela por meio de um código JavaScript.

## Objetivo
Demonstrar o uso básico de:
- Estrutura HTML
- Estilização CSS simples
- Função JavaScript acionada por evento (clique no botão)

## Estrutura do Código

| Parte | Descrição |
|------|-----------|
| <head> | Define informações da página, título e estilo. |
| <body> | Contém o conteúdo visível ao usuário (texto e botão). |
| <script> | Possui a função que exibe a mensagem quando o botão é clicado. |

## Funcionamento
1. A página é carregada no navegador.
2. O usuário clica no botão.
3. A função mostrarMensagem() é chamada.
4. O navegador exibe um alerta com um texto.

## Tecnologias Utilizadas
- *HTML* para estrutura
- *CSS* simples embutido no próprio arquivo
- *JavaScript* para interação
